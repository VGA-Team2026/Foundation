# spec
このフォルダ以下には、コードを記載するうえでの詳細な仕様書が格納されます

# フォルダ構成
## code
コードに関する仕様書が入ります  
プログラマーが記載します  

## gamedesign
ゲーム内の仕様、レベルデザインにまつわる仕様書が入ります  
プランナーが記載します  

## rule
AIの行動ルール、コード生成、ゲーム全体にかかわるルールが記載されます  
プランナー・プログラマーが記載します  

## ubidic
ゲーム内で使用する単語を説明する辞書です  
プランナー・プログラマーが記載します  

## usecase
ゲーム内のふるまいを定義します  
プランナー・プログラマーが記載します  

## ux
ゲームが目指す目標、ゴールをここに示します
