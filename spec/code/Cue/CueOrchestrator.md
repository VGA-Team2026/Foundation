# CueSequencerクラス設計

# 概要
- ゲーム中に行われる一連の演出をまとめてシーケンシャルに実行する
- シーケンシャルな指定はタイムラインを使用して行われる


# 必要要件
- タスクの発行と完了までを管理する
- シングルトンである
- Facade的な役割を担う


# 実装
- タイムライン処理から自動生成されたCueSequenceの派生クラスを外部命令で再生する
- タスクの実行中に競合する処理を検知する
	- 対応は今のところしない。これはTODOとしておく。


# 外部インタフェース
- Play: コールバックを引数に取り、渡されるCueSequenceを再生する。
- PlayAsync: async関数として定義する。対応するCueSequenceを再生する
※CueSequenceは所持しておき、再生状況を監視する。再生終了を確認したらリストから外す。
