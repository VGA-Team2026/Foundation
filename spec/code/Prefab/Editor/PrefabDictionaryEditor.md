# PrefabDictionaryEditorクラス設計

## コードの運用仕様
- PrefabDictionaryと対応するEditor拡張プログラム

## 実装
- Editorを継承する
- prefabDicListを追加しやすくする
- 「追加」ボタンを用意する
- バリデーションを実装する
	- バリデーションに引っかかった際は、エラーを出力する
	- エラーはCalloutで表示する
- バリデーションは保存ボタンの下、最後に表示する


## 処理フロー
## インスペクタ表示時
1. PrefabDictionaryのprefabDicListリストを取得する
2. 追加用の入力フィールド、追加ボタンを用意する

## 「追加」押下時
- {バリデーション}を実行する
	- エラーがある場合は以下の処理を行う
		1. 「エラーを解決してください」ダイアログを表示する
		2. 以後の処理は行わずに終了する
- 入力されたデータをprefabDicListに入れる

## 「保存」押下時
1. {バリデーション}を実行する
	1. エラーがある場合は以下の処理を行う
		1. 「エラーを解決してください」ダイアログを表示する
		2. 以後の処理は行わずに終了する
2. 保存する
3. アセットを上書き更新する

# バリデーション
- keyNameの重複がないかどうかを確認する。ある場合はエラーとする。
- (追加する際のみ)入力されていない値がないか確認する。ある場合はエラーとする。
