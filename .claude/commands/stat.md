# 現在の状態表示コマンド

現在のセッション状態を表示します。

## 処理フロー

1. **作業ディレクトリを表示**
   - `pwd` の結果を表示

2. **Gitブランチを表示**
   - `git branch --show-current` の結果を表示
   - `git status --short` で未コミット変更の有無を表示

3. **コンテクスト上の作業を表示**
   - 現在のTodoリストの内容を表示（あれば）
   - 直近で取り組んでいたタスクの概要を表示

4. **表示フォーマット**

以下のフォーマットで表示すること：

```
## 現在の状態

| 項目 | 値 |
|------|-----|
| 作業ディレクトリ | {pwd結果} |
| ブランチ | {branch名} |
| 未コミット変更 | {あり/なし（件数）} |

### コンテクスト上の作業
{Todoリストまたは直近タスクの概要}
```

## 注意事項
- 情報収集のみ行い、ファイル変更は行わない
- 簡潔に表示する
